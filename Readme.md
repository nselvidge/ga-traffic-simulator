#ga-traffic-simulator

## Description

A CLI to simulate traffic to google analytics

## Usage

To install ga-traffic-simulator from npm, run:

```
$ npm install -g ga-traffic-simulator
```

```node ./bin/ga-traffic-simulator --help```
